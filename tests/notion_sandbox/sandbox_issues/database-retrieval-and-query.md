# Database retrieval and query endpoints

The sandbox needs endpoints for retrieving a database and querying its pages.

## Endpoints needed

- `GET /v1/databases/{database_id}` — retrieve a database object
- `POST /v1/databases/{database_id}/query` — query pages in a database
